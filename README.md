# Screen Saver

A browser-based screen saver with multiple visual themes. Built with vanilla HTML, CSS, and JavaScript.

**[Launch Screen Saver](https://screen-saver.surge.sh/)**

## Themes

| Theme | Description |
|---|---|
| **Matrix** | Falling green katakana and alphanumeric characters (Matrix-style rain) |
| **Clock** | Terminal-styled digital clock with Unix timestamp, hex, and binary representations |
| **Time Elapsing** | Cyan count-up timer showing elapsed time since session start with particle effects |
| **Flip Clock** | Realistic flip-card clock with animated digit transitions and day/date display |

## Usage

1. Open the page and click a theme to launch fullscreen.
2. Press **ESC** or exit fullscreen to return to the menu.

## Deploy

```bash
surge .
```
